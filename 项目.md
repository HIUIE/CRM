# SmartTrade AI CRM 项目概览与完善计划

## 1. 项目核心框架
本系统是专为外贸 B2B 场景打造的智能化 ERP/CRM 平台。

*   **前端**: `React 19` + `Vite` + `Tailwind CSS 4` + `Motion`。
*   **后端**: `Express` + `SQLite` (全栈集成架构)。
*   **AI 引擎**: `Google Gemini` (`@google/genai`)。
*   **安全**: `JWT` + `HttpOnly Cookies` + `Bcrypt`。

## 3. 完善步骤 (Future Roadmap)

### 🚀 第一阶段：体验与性能优化 (短期)
1.  **架构解耦**: 拆分 `App.tsx` 的巨型组件，引入 `react-router-dom` 实现多路由管理。
2.  **搜索增强**: 实现全局模糊搜索，支持订单、客户、追踪号的跨模块查找。
3.  **多语言 (i18n)**: 完善中英双语界面，满足国际化业务需求。

### 🧠 第二阶段：AI 深度赋能 (中期)
1.  **多模态文件解析**: 支持直接上传 PDF 形式发票或图片，AI 自动录入系统。
2.  **智能获客辅助**: 根据客户交易史，AI 自动生成个性化的跟进邮件或产品推荐。
3.  **自动报表生成**: 一键导出财务对账单 (PDF/Excel) 及业务员业绩排行榜。

### 🛡️ 第三阶段：系统稳健性 (长期)
1.  **权限系统 (RBAC)**: 实现精细化角色权限控制（如财务仅看财务模块）。
2.  **库存管理联动**: 与仓库/工厂端实现库存实时同步预警。
3.  **单元与集成测试**: 引入 Vitest 与 Playwright，确保核心业务逻辑 100% 正确。
